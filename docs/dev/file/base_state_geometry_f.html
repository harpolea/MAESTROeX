base_state_geometry.F90¶

base_state_geometry_module¶

Description

a module for storing the geometric information so we don’t have to pass it

This module provides the coordinate value for the left edge of a base-state zone (r_edge_loc) and the zone center (r_cc_loc). As always, it is assumed that the base state arrays begin with index 0, not 1.
